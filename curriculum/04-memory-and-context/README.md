# Module 04: Memory & Context

## What You'll Learn
- How Claude Code's memory system works across sessions
- The different memory layers and when to use each
- Context management strategies that prevent quality degradation
- Building a persistent knowledge system

## Prerequisites
- Completed Modules 00-03
- A bootstrapped project you've worked in for at least 2-3 sessions

## Lessons
1. [How Memory Works](01-how-memory-works.md)
2. [The Memory Layers](02-memory-layers.md)
3. [Context Management](03-context-management.md)
4. [Building Persistent Knowledge](04-persistent-knowledge.md)

## Key Takeaway
Memory is what turns Claude Code from a tool you re-teach every session into a partner that knows your project, your preferences, and your patterns. Context management is how you keep it sharp throughout long sessions.
