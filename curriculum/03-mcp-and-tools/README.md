# Module 03: MCP & Tools

## What You'll Learn
- What MCP (Model Context Protocol) is and why it matters
- How to connect external tools to Claude Code
- Practical MCP servers you can use today
- Building your own tool ecosystem

## Prerequisites
- Completed Modules 00-02
- A bootstrapped project with CLAUDE.md

## Key Takeaway
MCP turns Claude Code from a coding tool into an integration layer. Every external system you connect makes Claude Code smarter for your specific work.
