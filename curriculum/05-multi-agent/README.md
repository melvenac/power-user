# Module 05: Multi-Agent Work

## What You'll Learn
- How Claude Code coordinates multiple agents
- When to use agents vs. doing it in one thread
- Task decomposition — breaking big work into parallel pieces
- Practical multi-agent patterns

## Prerequisites
- Completed Modules 00-04
- Comfortable with Claude Code's tool system and context management

## Lessons
1. [Why Multi-Agent?](01-why-multi-agent.md)
2. [The Agent Tool](02-agent-tool.md)
3. [Task Decomposition](03-task-decomposition.md)
4. [Agent Teams](04-agent-teams.md)

## Key Takeaway
Multi-agent work is Claude Code's answer to complex tasks. Instead of cramming everything into one context window, split the work across specialized agents that run in parallel.
